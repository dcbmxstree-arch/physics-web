📘 Conceptos Clave

Determinante y Rango: El determinante indica si una matriz tiene inversa. El rango es el número de filas independientes tras la reducción. Si Rango(A) < n, el sistema no tiene solución única.
Método de Gauss-Jordan: Consiste en aplicar operaciones de fila (intercambio, escalamiento y sustitución) para transformar [A|B] en [I|x].
Tipos de Solución: Determinado (única), Indeterminado (infinitas) o Incompatible (sin solución si aparece una fila 0 = constante).
